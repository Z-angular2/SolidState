Lightning Talk

The Coverpage of my lightingtalk presentation

I gave a presentation on NPM & GIT shortcuts it was fun, i was a bit nervous at first but as I went on and the nervousness lost it's edge on me. I wanted to give this talk because it's always new developers coming into the Developer Ecosystem & even some senior developers could use some shortcuts to help with their DEV workflow

SanDiego AngularMeetup Tweet

The San Diego Angular Meetup Tweeting to let Developers know it's on! Im Happy to see my twitter handle up there.

So I started out with the NPM section of my talk, went through a few very time efficent commands that are favorites of some of the industry pros like John Papa (a well known Angular Enthusiats) and some of the developers on egghead.io use them so yea I displayed them along with another time saver which is the npm init -y because if you're not using it, you have to sequentially go down the line pressing enter, but with this gem, it default all the inquiries to yes by default & if you need to make a change to it, just pop open your package.json folder, but the command is really useful for initializing apps

Next up is the Git section of my talk

Okay so in the section is about GIT ALIASES, oh sweet git aliases I love how time efficient you are, and I plan to keep taking you for granted in the terminal, ahh... Okay back to the presentation. So I start with how to get these puppies actually working then I go on to show how to set them up quickly.
